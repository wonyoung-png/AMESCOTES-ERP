# 발주/주문 관리

## 개요
해외 거래처(중국, 대만, 호주) 발주 및 국내 주문 관리.

## 주요 기능
- 발주 등록/수정/조회 (음성 등록 포함)
- 입고 미처리 발주 자동 감지 → 담당자 알림
- 거래처별 발주 현황 조회
- 납기일 기반 우선순위 관리

## 누락 알림 조건
- 발주 후 예정 입고일 경과했으나 입고 미처리인 경우
- 발주 등록 후 승인 대기 중인 건이 3일 이상 경과한 경우

## 음성 빠른 등록 예시
> "중국 거래처에 레더 핸들 500개 발주해줘, 납기는 4월 15일"

## API 연동 설계 (예정)
```
GET  /api/orders                 # 발주 목록
POST /api/orders                 # 발주 등록
PUT  /api/orders/{id}            # 발주 수정
GET  /api/orders/pending         # 입고 미처리 발주 목록
GET  /api/orders/{id}/status     # 발주 진행 상태
```
