# 배송/납기 관리

## 개요
해외 배송 및 납기 일정 관리 (주요 거래처: 중국, 대만, 호주).

## 주요 기능
- 납기 일정 등록/조회
- 납기 임박 건 자동 알림 (D-7, D-3, D-1)
- 배송 진행 상태 추적
- 납기 준수율 통계

## 누락 알림 조건
- 납기일 7일 전 미출고 상태인 경우
- 배송 등록 후 추적 정보 없는 경우

## API 연동 설계 (예정)
```
GET  /api/shipping               # 배송 현황
POST /api/shipping               # 배송 등록
GET  /api/shipping/upcoming      # 납기 임박 목록 (7일 이내)
GET  /api/shipping/overdue       # 납기 초과 목록
```
