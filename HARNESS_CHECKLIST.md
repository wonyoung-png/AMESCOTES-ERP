# AMESCOTES ERP — 하네스 검수 체크리스트

## 작업 지시 템플릿 (올리브 → 에이스)
```
[작업명]: 
[요구사항]:
  1.
  2.
  3.
[완료 기준]: 빌드 성공 ✅ + 모든 항목 ✅
[금지사항]: 사전원가 수정 금지
[베라 검수]: 필수
```

---

## 1차 검수 — 에이스 자체 (작업 완료 후)
- [ ] 요구사항 항목별 구현 완료
- [ ] `npx vite build` 성공
- [ ] 사전원가 파일 미변경
- [ ] 삭제 confirm 팝업 있음
- [ ] Supabase 저장 확인

## 2차 검수 — 아리아 교차 (에이스 완료 후)
- [ ] 루멘 관련 화면 정상 작동
- [ ] 데이터 표시 정상
- [ ] UI 깨짐 없음

## 3차 검수 — 베라 독립 (최종)
- [ ] 요구사항 100% 충족 여부
- [ ] 빌드 성공 재확인
- [ ] 사전원가 레드라인 확인
- [ ] git push 완료
- [ ] **통과 → 올리브에게 보고 / 미통과 → 재작업 요청**

---

## 가비지 컬렉션 체크리스트 (매주)
- [ ] 사용하지 않는 import 제거
- [ ] 주석 처리된 코드 블록 검토
- [ ] 콘솔 로그 디버그 코드 제거
- [ ] 번들 크기 확인 (현재 2.4MB → 목표 1.5MB 이하)
- [ ] 미사용 컴포넌트 파일 확인

---
